# Fase 3 - Prototipo UX Discord

## Equipo
UXploradores

## Materia
Interacción Humano Computadora

## Descripción
Este repositorio contiene un prototipo frontend hardcodeado para la propuesta de mejora de usabilidad del sistema *Discord*. Este sistema no busca ningún lucro y solo busca cumplir con las indicaciones dada en la clase dictada en la universidad.

El prototipo está basado en los problemas identificados durante la evaluación heurística de la Fase 2.

## Objetivo
Diseñar y validar soluciones concretas a violaciones de heurísticas de usabilidad mediante un prototipo funcional de baja/media fidelidad.

## Tecnologías
- HTML
- CSS
- JavaScript

## Alcance
El prototipo no implementa lógica de negocio real, backend, autenticación ni persistencia de datos.
Su objetivo es representar flujos de interacción mejorados para pruebas con usuarios.

## Escenarios incluidos
1. Unirse a un servidor
2. Enviar mensajes directos
3. Entrar a un canal de voz
4. Silenciar notificaciones
5. Verificación de usuario

## Problemas abordados
- Alta carga cognitiva
- Falta de claridad en la navegación
- Sobrecarga visual
- Falta de feedback inmediato
- Accesos accidentales
- Configuraciones dispersas
- Verificación poco clara

## Propuestas de mejora
- Buscador claro para unirse a servidores
- Indicadores de estado en mensajes directos
- Confirmación antes de entrar a canales de voz
- Centro único de notificaciones
- Flujo guiado de verificación

## Estado del proyecto
En desarrollo.
