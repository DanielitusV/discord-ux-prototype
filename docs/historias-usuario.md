# Historias de usuario - Fase 3

## 1. Unirse a servidor
Como usuario nuevo, quiero encontrar servidores fácilmente, para no depender de enlaces externos.

Como usuario, quiero entender claramente dónde unirme, para no confundirme con el botón “+”.

## 2. Mensajes directos
Como usuario, quiero saber si mi mensaje fue leído, para tener feedback claro.

Como usuario, quiero ver el estado del contacto, para saber si está disponible.

Como usuario, quiero evitar cerrar el chat accidentalmente, para no perder lo que estoy escribiendo.

## 3. Canal de voz
Como usuario, quiero confirmar antes de entrar a un canal de voz, para evitar accesos accidentales.

## 4. Notificaciones
Como usuario, quiero configurar notificaciones en un solo lugar, para no perder tiempo navegando.

## 5. Verificación
Como usuario nuevo, quiero una guía paso a paso, para completar la verificación sin confusión.
